# 167. Two Sum II - Input Array Is Sorted

## 我的思考
有序数组 + 两数和，直接指向双指针。

双指针之所以可以安全使用，本质上是一种基于有序性的剪枝。  
朴素做法是枚举每一对可能的值，但“有序”提供了降维的可行性。

当前最小值加当前最大值：
- 如果偏大，说明这个最大值已经失去价值，不可能再匹配成功，可以直接丢弃
- 如果偏小，说明这个最小值已经失去价值，不可能再匹配成功，可以直接丢弃

因此每一步都能稳定排除一部分不可能解，最终在线性时间内找到答案。

## 关键点
1. 有序数组中的两数和问题，优先想到双指针。
2. 双指针的本质是利用单调性做剪枝，而不是暴力枚举。

## 复杂度
- 时间：`O(n)`
- 空间：`O(1)`

## 注意
1. 二分做法也能写，但不是这题最自然的解法。
2. 这题最关键的识别是：**有序 + two sum = two pointers**。
