# 1293. Shortest Path in a Grid with Obstacles Elimination

## 我的思考
本题直接识别为无权图最短路，使用 BFS。  
与普通二维 BFS 不同，本题状态需要额外携带已消耗的障碍数量。  
`visited[i][j]` 不使用 boolean，而是记录到达 `(i, j)` 时的最小障碍消耗；若新的消耗不小于历史最优，则该状态没有继续扩展价值。

## 卡点

## 关键点
- 带资源限制的最短路，状态不能只看位置，需要加入资源维度。
- `visited` 可以记录“到达该位置的最优资源消耗”，不一定是 boolean。
- 同一位置下，障碍消耗更少的状态支配障碍消耗更多的状态。

## 复杂度
- 时间：`O(mn * min(k, mn))` 的宽松上界；当前剪枝写法中每个格子只在消耗降低时入队，实际通常更低。
- 空间：`O(mn)`，用于 `visited` 和队列。
